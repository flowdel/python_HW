# KMER SPECTRUM

### Required arguments

* ```-f``` - fastq file
* ```-k``` - kmer size
* ```-q``` - minimal quality of each nucleotide

### Running

Open terminal and run:
```
hw3.py -f seq.fastq -k 15 -q 20
```
